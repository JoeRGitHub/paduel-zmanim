# לוחות זמני תפילה - פדואל

אתר לניהול והצגת לוחות זמני תפילה לקהילת פדואל.

## תכונות

- 📅 **לוח זמנים נוכחי** - תצוגה אינטראקטיבית של זמני התפילה
- 📚 **ספריית לוחות** - ארכיון של כל לוחות הזמנים (קיץ, חורף, תשרי)
- 🔍 **חיפוש וסינון** - מציאת לוחות לפי עונה, שנה ומילות חיפוש
- 🔐 **ממשק ניהול** - העלאה ועריכת לוחות זמנים (למנהל בלבד)
- 📱 **תמיכה בנייד** - עיצוב רספונסיבי לכל המכשירים
- ✡️ **תמיכה מלאה בעברית** - RTL וממשק בעברית

## מבנה הפרויקט

```
paduel-zmanim/
├── index.html          # עמוד הבית - לוח זמנים נוכחי
├── library.html        # ספריית לוחות זמנים
├── admin.html          # ממשק ניהול (דורש אימות)
├── data/
│   └── pdfs.json      # מטא-דאטה של כל הלוחות
├── pdfs/              # תיקיית קבצי PDF
│   └── summer-5786.pdf
└── js/
    ├── pdf-manager.js # ניהול תצוגת הלוחות
    └── admin.js       # לוגיקת הניהול

```

## התקנה והפעלה

### אופציה 1: פתיחה מקומית

```bash
# פשוט פתח את הקבצים בדפדפן
open index.html
```

### אופציה 2: שרת פיתוח מקומי

```bash
# עם Python
python -m http.server 8000

# עם Node.js
npx http-server

# גש ל- http://localhost:8000
```

### אופציה 3: GitHub Pages (מומלץ)

1. העלה את הקבצים ל-GitHub repository
2. הפעל GitHub Pages ב-Settings → Pages
3. האתר יהיה זמין בכתובת: `https://username.github.io/paduel-zmanim`

## שימוש במערכת

### למשתמשים רגילים

1. **צפייה בלוח נוכחי**: גש ל-`index.html` או לעמוד הבית
2. **עיון בארכיון**: לחץ על "ספרייה" לצפייה בכל הלוחות
3. **חיפוש**: השתמש בסינון לפי עונה, שנה או חיפוש חופשי
4. **צפייה ב-PDF**: לחץ על "צפייה" לפתיחת הלוח בדפדפן
5. **הורדה**: לחץ על "הורדה" לשמירת הקובץ

### למנהלים

#### כניסה למערכת הניהול

1. גש ל-`admin.html`
2. התחבר עם:
   - **שם משתמש**: `admin`
   - **סיסמה**: `paduel2026`

#### הוספת לוח זמנים חדש

1. מלא את פרטי הלוח:
   - כותרת (לדוגמה: "לוח זמני תפילה לקיץ התשפ״ו")
   - עונה (קיץ / חורף / תשרי)
   - שנה עברית במספרים (5786)
   - שנה עברית בעברית (תשפ״ו)
   - העלה קובץ PDF

2. לחץ "העלה לוח זמנים"

3. בצע את ההוראות שיופיעו:

   ```bash
   # 1. שמור את קובץ ה-PDF בתיקייה
   cp downloaded-file.pdf pdfs/summer-5786.pdf

   # 2. הורד ושמור את pdfs.json המעודכן
   # (לחץ על הקישור שמופיע במסך)
   mv ~/Downloads/pdfs.json data/pdfs.json

   # 3. בצע commit ל-Git
   git add pdfs/summer-5786.pdf data/pdfs.json
   git commit -m "הוספת לוח זמנים: קיץ תשפו"
   git push
   ```

#### עריכת לוח קיים

1. לחץ על "✏️ ערוך" ליד הלוח הרצוי
2. הטופס יתמלא מראש
3. ערוך את השדות ולחץ "העלה לוח זמנים"
4. בצע את ההוראות המעודכנות

#### מחיקת לוח

1. לחץ על "🗑️ מחק" ליד הלוח הרצוי
2. אשר את המחיקה
3. בצע את הפעולות שיופיעו:
   ```bash
   git rm pdfs/summer-5786.pdf
   git add data/pdfs.json
   git commit -m "מחיקת לוח זמנים"
   git push
   ```
